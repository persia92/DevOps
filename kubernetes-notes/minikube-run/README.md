### Minikube- Commands

- Before starting minikube
```sh
eval $(minikube docker-env)
```
- start minikube
```sh
minikube start
```
- stop minikube 
```sh
minikube stop
```

# Install kubectl https://kubernetes.io/docs/tasks/tools/install-kubectl-linux/
```sh
kubectl get pods
kubectl create -f x.yaml
```
